# GreenGrow Project

## Overview
GreenGrow is a responsive web application designed to promote eco-friendly practices and provide a platform for users to manage their green initiatives. The application features a clean and modern interface with smooth animations and a nature-inspired background.

## Features
- **User Authentication**: Users can register and log in to access their accounts.
- **Admin Dashboard**: Admins can monitor users, verify sellers, and manage passwords.
- **Responsive Design**: The application is fully responsive, ensuring a seamless experience on all devices.
- **Smooth Animations**: CSS animations enhance the user experience with engaging transitions.

## Setup Instructions
1. Clone the repository:
   ```
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```
   cd GreenGrow
   ```
3. Install dependencies (if any):
   ```
   npm install
   ```
4. Open `src/index.html` in your web browser to view the application.
